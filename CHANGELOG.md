## 功能特性

### 核心功能
- 🤖 每天 UTC 0点自动运行
- 📁 按日期归档文章
- 📊 自动更新 README
- 🌱 自动提交到 GitHub
- 🔄 网络请求自动重试（最多3次）
- ⏱️ 请求超时保护（10秒）

### 技术特性
- 🐍 Python 3.7+ 支持
- 📦 最小依赖（仅 requests）
- 🔧 模块化设计
- 📝 完整的文档
- 🎯 清晰的错误处理
